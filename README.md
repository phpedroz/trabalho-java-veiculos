## Objetivo

O objetivo do sistema de veículos é organizar e gerenciar informações de veículos de forma simples e prática.

Este projeto serve como base para estudos de lógica de programação, organização de dados e desenvolvimento de sistemas.

---

## Funcionalidades previstas

- Cadastro de veículos  
- Listagem de veículos  
- Edição de informações dos veículos  
- Remoção de veículos

---

## Tecnologias utilizadas

- Linguagem: **Java**
- Banco de dados: **Arquivo TXT**
- Interface: **Console (menu interativo)**
